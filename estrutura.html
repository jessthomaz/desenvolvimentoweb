Estrutura básica do HTML
Para iniciantes

O HTML possui uma estrutura básica, vou deixar disponível no final.
Head é para declarações e Body para conteúdo visual.
Head e Body estão na mesma hierarquia.

Além disso existem Tags como P (parágrafo), H1 (título), EM (itálico), Strong (negrito), br (quebra de linha), entre outros.
Caso você queira criar um espaço em branco não quebrável, basta usar &NBSP; (minúnculo), pode ser usado para acrescentar vários espaços.

Também existem parâmetros como Style, Charset, Name, Content, Href, entre outros.
O CSS pode está incluso no HTML ou simplesmente separado em um documento próprio para o CSS.
Em uma página WEB, geralmente é usado HTML, CSS e JavaScript.
